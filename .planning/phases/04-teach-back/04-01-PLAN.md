Create the `api/` directory and `api/evaluate.js`. This is a Vercel serverless function. Use the exact pattern from 04-RESEARCH.md "Full api/evaluate.js Handler" section:

- Import `Anthropic` from `@anthropic-ai/sdk`
- Create client with `new Anthropic()` (reads `ANTHROPIC_API_KEY` from env automatically)
- Export default async handler function receiving `(req, res)`
- Reject non-POST with 405
- Destructure `{ moduleId, conceptArea, explanation }` from `req.body ?? {}`
- Validate `explanation?.trim()` is truthy, return 400 if not
- Call `client.messages.create()` with model `"claude-haiku-4-5"`, max_tokens 512
- System prompt (the encouraging ADHD-tutor prompt from research — score 1-5, pass boolean, correct/missed arrays, plain English bullets, encouraging tone)
- User message: `Concept area: ${conceptArea ?? "General"}\n\nExplanation: ${explanation}`
- Parse response with defensive JSON extraction: find first `{` in raw text before parsing
- Return 200 with parsed evaluation JSON on success
- Return 500 with `{ error: "Evaluation failed" }` on any catch

Code style: ES module syntax (`import`/`export`), double quotes, no semicolons.
