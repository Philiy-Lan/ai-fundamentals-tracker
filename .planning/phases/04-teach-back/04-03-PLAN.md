  <name>Task 2: End-to-end Teach-Back verification</name>
  <files></files>
  <action>
Human verifies the complete Teach-Back activity end-to-end in a real browser. This checkpoint covers all behaviors that cannot be tested in jsdom: Web Speech API voice recording, live Vercel API route, real navigator.onLine toggling, and cross-browser fallback behavior.

**Prerequisites:**
- Ensure `ANTHROPIC_API_KEY` is set as a Vercel environment variable
- Deploy to Vercel (or run `vercel dev` locally if the Vercel CLI is installed)

**Test 1 — Text input flow (any browser):**
1. Open the app and navigate to any module (e.g., Module 1)
2. Tap the "Teach-Back" activity to expand the panel
3. Verify a concept prompt appears at the top (should be one of the module's prompts)
4. Type an explanation in the textarea
5. Tap Submit
6. Verify loading state appears ("Evaluating...")
7. Verify structured feedback appears: numeric score (1-5), bullet points for correct items, bullet points for missed items
8. If score >= 3 (pass): verify the activity is auto-marked complete
9. If score < 3 (fail): verify "Try Again" button appears, tap it, verify you return to the input screen

**Test 2 — Voice input (Chrome or Edge only):**
1. Open the app in Chrome
2. Navigate to a module, tap Teach-Back
3. Tap the mic button — verify pulsing animation starts
4. Speak an explanation
5. Tap the mic button again to stop
6. Verify transcript appears in an editable textarea
7. Optionally edit the transcript
8. Tap Submit and verify feedback appears

**Test 3 — Text fallback (Firefox):**
1. Open the app in Firefox
2. Navigate to a module, tap Teach-Back
3. Verify NO mic button appears
4. Verify a textarea is shown as the primary input
5. Type and submit — verify same feedback flow

**Test 4 — Offline fallback:**
1. Open Chrome DevTools, go to Network tab, check "Offline"
2. Navigate to a module, tap Teach-Back
3. Verify "Needs internet connection" message appears instead of input UI
4. Uncheck "Offline" — verify the input UI appears

**Test 5 — Different modules:**
1. Check Module 1, Module 4, and Module 8 teach-back prompts
2. Verify each shows a different module-specific concept prompt
3. Refresh the page on the same module a few times — verify prompt may change (random selection)
