    **Rating buttons (D-08, FLASH-02):** Only render when `isFlipped === true`. Two buttons:

    - "Missed it" with XCircle icon — calls `handleRate(false)`
    Use `data-testid="btn-correct"` and `data-testid="btn-incorrect"` for test targeting.

    **handleRate callback (D-09, D-12, FLASH-03, FLASH-06):**
    ```javascript
    const handleRate = useCallback((wasCorrect) => {
      if (wasCorrect) setScore(s => s + 1)
      const nextIndex = currentIndex + 1
      if (nextIndex >= cards.length) {
        if (!completedRef.current) {
          completedRef.current = true
          onComplete()
        }
      } else {
        setCurrentIndex(nextIndex)
        setIsFlipped(false)
      }
    }, [currentIndex, cards.length, onComplete])
    ```

    **Score display (D-09, FLASH-03):** Render `{score} / {cards.length} correct` in the header area.

    **Progress indicator (D-10, FLASH-04):** Render `Card {currentIndex + 1} of {cards.length}` in the header area.

    **Shuffle toggle (D-11, FLASH-05):** Button with Shuffle icon. On click:
    - If not shuffled: Fisher-Yates shuffle a copy of `source`, set as `cards`, reset `currentIndex` to 0, reset `isFlipped` to false, set `isShuffled(true)`.
    - If already shuffled: Restore `source` order, reset index, set `isShuffled(false)`.
    Use `data-testid="btn-shuffle"` for test targeting.

    Fisher-Yates inline helper:
    ```javascript
    function shuffleArray(arr) {
      const copy = [...arr]
      for (let i = copy.length - 1; i > 0; i--) {
        const j = Math.floor(Math.random() * (i + 1))
        ;[copy[i], copy[j]] = [copy[j], copy[i]]
      }
      return copy
    }
    ```
